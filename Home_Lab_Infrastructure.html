Geovanna Orlando

Purpose

Multi-VM environment built to support hands-on cybersecurity coursework and security tool validation. Configured with isolated networking to practice vulnerability scanning, malware analysis workflows, and security concepts in a controlled setting.

Environment Overview

Platform Oracle VirtualBox
Host Windows 11 Pro | 40 GB RAM | 1 TB SSD
Network Setup Dual Adapters: NAT (outbound) + Internal Network (192.168.10.0/24)
Purpose Safely replicate security workflows without impacting external networks

Virtual Machines

  • Kali Linux 2024.3 — Attacker VM for reconnaissance, scanning, exploitation, and packet capture
  • Ubuntu Server 22.04 LTS — Internal host for web services, SSH access, and vulnerability testing
  • Windows 11 Evaluation — Endpoint for event log monitoring, patch testing, and remediation practice
  • Metasploitable 2 — Intentionally vulnerable target system for ethical exploitation and learning

Network Topology

  • Segmentation: NAT used for outbound updates only
  • Isolation: No connection between Internal Network and host LAN
  • Monitoring: Network traffic captured for analysis using Wireshark
